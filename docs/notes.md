## USB mass storage class example jumper connections (MSP-EXP4305529LP LaunchPad board):

| micro-SD pin | LaunchPad pin |
| ------------ | ------------- |
| CS           | P3.7          |
| SCLK         | P4.3          |
| DO           | P4.2          |
| DI           | P4.1          |
